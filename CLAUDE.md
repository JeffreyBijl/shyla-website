# Shyla Website — fit.foodbyshyla

## Project

Voedingscoach Shyla's website. Wordt gemigreerd van Vite SPA naar Astro SSG voor SEO/GEO.

- **Taal:** Nederlands (NL)
- **Hosting:** GitHub Pages (`jeffreybijl.github.io/shyla-website/`)
- **CMS:** GitHub API-based admin panel (standalone SPA bundle)
- **Data:** JSON bestanden (`recipes.json`, `blog.json`)

## Conventies

- Vanilla TypeScript, geen frameworks (React/Vue) tenzij expliciet gevraagd
- CSS: bestaande custom properties en design system behouden
- Fonts: Playfair Display (headings), Inter (body)
- Alle content in het Nederlands
- Geen emoji in code/comments tenzij gevraagd

## Autonomie

- Maak bestanden aan, bewerk en verwijder zonder bevestiging te vragen
- Voer builds en tests uit zonder bevestiging
- Installeer npm packages als dat nodig is voor de taak
- Commit NIET automatisch — alleen op verzoek
- Push NIET automatisch — alleen op verzoek
